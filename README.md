# 🚀 Smart File Organizer (Python Automation Project)

## 📌 Overview

Smart File Organizer is a beginner-friendly Python automation project that helps in organizing messy directories by automatically sorting files into categorized folders such as Images, Documents, and Videos.

This project is a first step towards learning real-world automation using Python.

---

## 💡 Objective

To automatically organize files into structured folders based on their file types, making directories clean and easy to navigate.

---

## 🔧 Tech Stack

* Python
* os module
* shutil module

---

## ✨ Features

* 📂 Automatically detects file types
* 🗂️ Creates folders if they don’t exist
* 🔄 Moves files into respective folders (Images, Docs, Videos)
* 🧹 Cleans and organizes messy directories

---

## 📁 How It Works

1. The script scans all files in the given directory
2. Identifies file types based on extensions
3. Creates folders (if not already present)
4. Moves files into their respective folders
